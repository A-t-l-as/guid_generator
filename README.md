# guid_generator
Mały program do generowania guid.
Po uruchomieniu od razu wygeneruje się plik guid.bin, w którym znajduje się guid binarnie. Natomiast plik guid.txt
zawiera guid w postaci tekstowej.

---

A small program for generating GUIDs. Very useful for files with GUIDs.
After launching, it will immediately generate a guid.bin file containing the GUID in binary form. The guid.txt file, on the other hand,
contains the GUID in text form.
